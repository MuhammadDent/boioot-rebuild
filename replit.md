# Boioot — Real Estate Platform

## Description
Boioot is an Arabic-first real estate marketplace + SaaS platform for the Syrian market.

## Tech Stack
- **Frontend**: Next.js (App Router, TypeScript)
- **Backend**: ASP.NET Core Web API (.NET 8)
- **Database**: SQL Server
- **Architecture**: Modular Monolith
- **UI**: RTL, Arabic-first, green branding (`#2E7D32`)
