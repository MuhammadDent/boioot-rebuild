# Boioot — منصة العقارات السورية

منصة عقارية متكاملة تجمع بين نموذج السوق (Marketplace) ونموذج SaaS.

## التقنيات

| الطبقة | التقنية |
|--------|---------|
| Frontend | Next.js (App Router, TypeScript) |
| Backend | ASP.NET Core Web API (.NET 8) |
| Database | SQL Server |
| Architecture | Modular Monolith |
| UI | RTL عربي أولاً |

## هيكل المشروع

```
boioot/
├── apps/
│   ├── frontend/     # Next.js App
│   └── backend/      # ASP.NET Core API
└── docs/             # التوثيق وخطة الإعداد
```

## الهوية البصرية

- اللون الأساسي: أخضر `#2E7D32`
- اتجاه: RTL (عربي أولاً)
- نمط: Marketplace بسيط وواضح

## حالة المشروع

في مرحلة الإعداد — لا يوجد كود بعد.
