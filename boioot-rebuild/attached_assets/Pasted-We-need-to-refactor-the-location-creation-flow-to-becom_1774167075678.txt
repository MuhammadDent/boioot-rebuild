### Soft Similarity Layer
In addition to strict normalize, implement soft similarity suggestion logic:
- detect very similar entries
- examples:
  - الهفوف / هفوف
  - البصيرة / بصيرة
  - دير الزور / ديرزور

If similar matches exist:
- DO NOT auto-create immediately
- return:
  {
    status: "similar",
    suggestions: [...]
  }

If exact normalized match exists:
- return:
  {
    status: "exists",
    item: existingItem
  }

If no exact or similar match:
- create new record and return:
  {
    status: "created",
    item: createdItem
  }

Support:
- forceCreate: true
But only after UI confirmation.
