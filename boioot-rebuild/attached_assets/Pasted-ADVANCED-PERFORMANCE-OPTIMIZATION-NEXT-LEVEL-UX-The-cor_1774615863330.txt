PHASE 1 — SKELETON UI (REPLACE SPINNERS)

========================================
PHASE 2 — IMAGE OPTIMIZATION
========================================

1) Use thumbnails in listings (NOT full images)

4) Add blur placeholder

========================================
PHASE 3 — PREFETCHING
========================================

When user hovers or scrolls near:

- property card → prefetch details
- next page → prefetch next page data

Example:
- prefetch property detail on hover
- prefetch next pagination page

========================================
PHASE 4 — PARALLEL UI LOADING
========================================

Split:
- main content loads first
- secondary widgets load after
