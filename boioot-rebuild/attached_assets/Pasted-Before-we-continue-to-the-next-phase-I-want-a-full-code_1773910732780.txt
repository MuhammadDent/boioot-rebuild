CODE REVIEW / VALIDATION TASK
========================

- Verify permission checks are implemented consistently

4) Staff / Roles architecture
- Verify separation between platform users and internal staff
- Verify roles/pages/sidebar logic is coherent
- Check for hardcoded assumptions that will break scalability

5) Blog SEO system review
- Verify SEO mode handling is correct:
  - Auto
  - Template
  - Custom
- Verify fallback logic is safe
- Verify template variable replacement is correct
- Verify no duplicated resolver logic in conflicting places

6) Data consistency / API contracts
- Verify frontend types match backend DTOs
- Verify no field mismatches exist
- Verify null/undefined handling is safe
- Verify forms correctly load and submit all required fields

7) Security / authorization
- Identify any authorization gaps

8) Code quality / maintainability
