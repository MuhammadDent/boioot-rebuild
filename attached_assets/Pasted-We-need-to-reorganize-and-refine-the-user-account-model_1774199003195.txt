==================================================
BUSINESS MODEL (CONFIRMED)
==================================================

We want to clearly separate between:

1. RealEstateOffice
2. DeveloperCompany
3. Broker
4. Agent
5. Owner
6. User / Seeker

--------------------------------------------------

A) RealEstateOffice
- Business entity (office)
- Can manage profile, listings, billing, messages, analytics
- CAN create and manage Agents

B) DeveloperCompany
- Business entity (developer)
- Can manage projects, units, billing, messages, analytics
- CAN create and manage Agents

C) Broker
- Independent professional (solo account)
- Has almost same operational capabilities as office
- BUT:
  - cannot create agents
  - cannot manage team

D) Agent
- NOT a public registration type
- Created ONLY by:
  - RealEstateOffice
  - DeveloperCompany
- Must belong to parent organization
- Permissions controlled by parent (RBAC)

E) Owner
- Individual property owner
- Can manage personal listings
- No team capabilities

F) User / Seeker
- Regular user browsing the platform
- Can browse, save, and inquire
